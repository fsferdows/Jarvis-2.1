# JARVIS Voice Command Examples

This document provides comprehensive examples of commands you can use with JARVIS, organized by category.

## üéØ Command Format

All voice commands should start with the wake word (default: "**Jarvis**"), followed by your request:

```
Jarvis, [your command here]
```

You can also type commands directly when JARVIS is listening.

## üí° Tips for Best Results

1. **Be Clear and Specific**: "Open Chrome" is better than "open browser"
2. **Use Natural Language**: JARVIS understands conversational speech
3. **Provide Context**: "Email John about the meeting" includes context
4. **Chain Actions**: You can give multi-step instructions
5. **Be Patient**: Wait for JARVIS to acknowledge before the next command

## üö´ Exit Commands

To shut down JARVIS:
- "Jarvis, quit"
- "Jarvis, exit"
- "Jarvis, shutdown"
- "Jarvis, goodbye"

Or press `Ctrl+C` to force quit.

## üîß Customization

You can customize the wake word and voice settings in the `.env` file:

```bash
WAKE_WORD="computer"    # Change to your preferred wake word
VOICE_RATE=200          # Adjust speaking speed
VOICE_GENDER="female"   # Change voice gender
```
